# Outputs Directory

## 📊 Visualizations & Results Storage

This directory will contain all output files generated by the notebook analysis.

### Expected Files:

#### 1. Exploratory Data Analysis (EDA)
- `01_class_distribution.png` - Bar chart of traffic sign classes
- `02_sample_images.png` - 10x4 grid of sample traffic signs
- `03_data_augmentation.png` - Before/after augmentation examples

#### 2. Model Training & Evaluation
- `04_detection_pipeline.png` - Example of detection + cropping stage
- `05_cnn_training_history.png` - Training/validation loss curves
- `06_speed_vs_accuracy_tradeoff.png` - Model comparison scatter plot
- `07_confusion_matrix.png` - 43×43 heatmap of predictions
- `08_sample_predictions.png` - Example predictions with confidence scores
- `09_end_to_end_pipeline.png` - Full pipeline demonstration

#### 3. Summary Reports
- `cnn_configurations.csv` - Hyperparameter configurations for 3 CNN variants
- `speed_vs_accuracy_tradeoff.csv` - Model performance metrics
- `classification_report.txt` - Precision, recall, F1-score for each class

## 🔄 How These Are Generated:

These files are automatically created when you run the Jupyter notebook:
```bash
jupyter notebook ADAS_Traffic_Sign_Pipeline.ipynb
```

Each cell in the notebook generates corresponding output files using:
- **Matplotlib/Seaborn** for visualizations
- **Python csv module** for tabular data
- **Scikit-learn** for classification metrics

## 📈 File Sizes (Approximate):
- PNG images: 200-500 KB each
- CSV files: 50-200 KB each
- TXT reports: 5-50 KB

**Total: ~5-10 MB after complete execution**
